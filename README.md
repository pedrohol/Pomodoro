# Pomodoro
Depois de utiliza um timer pomodoro na minha rotinha de estudo, eu pensei, por que não fazer o meu próprio aplicando o que estudei até agora.

## Features
O aplicativo consiste em um timer no centro da tela, controlado por um botão com 3 funcionalidades, e um botão para pular de ciclo. 

**Start | Pause | Resume**

- Inicia, Pausa, e Continua o timer do momento em que foi pausado.

**Skip Buttom**

- Pula para o proximo ciclo.

## Pomodoro Cycle
O aplicativo possui 3 diferentes ciclos. Assim que o temporizador chega a zero, o novo ciclo se inicia. 

**Pomodoro**

- Um timer de 30 minutos. Iniciando um ciclo Pomodoro.

**Short Break** 

- Um timer de 5 minutos. Carregado ao fim de um ciclo Pomodoro

**Long Break**

- Um timer de 15 minutos, Carregado ao fim de 4 ciclos Pomodoro.
